コンテンツ利用規約

以下、豊穣ミノリ / 豊穣折紙を「私」と表記します。

投稿コンテンツの使用・引用

引用元表記のない転載は禁止します。 私が投稿したコンテンツを使用する場合、引用元表記として以下のいずれか1つ以上を分かる場所に表記してください。

  • 豊穣ミノリ
  • 豊穣折紙
  • Hojo Minori
  • ハッシュタグ #豊穣折紙 #豊穣ミノリ #豊穣配信 #ミノリアート

ただし、 GitHub 上にライセンス表記がある場合はそちらを優先します。

営利目的(広告収入など、間接的なものを含む)で利用・引用する場合は必ず事前にメールでご連絡ください(連絡先はこのページ下部に記載)。 無許可での営利目的利用・引用は禁止します。
非営利目的で利用・引用する場合は、連絡いただく必要はありません。 (ただし、個人的に気になるので教えていただけると嬉しいです)

二次創作

イラスト、折り紙作品など何でも大歓迎です！
豊穣ミノリ(キャラクター)は基本的に中性的なキャラクターとして描いていただけると嬉しいです。
原則として、二次創作作品の著作権は二次創作者に帰属します。二次創作物を利用したい場合、二次創作者にご確認ください。

作品を Twitter(現X), Instagram に投稿する場合、以下のハッシュタグを使用してくださると嬉しいです。定期的にチェックしています。

  • 折り紙作品 … #豊穣折紙 (創作者表記を兼ねます)
  • 折り紙作品以外 … #ミノリアート または #豊穣ミノリ

折り紙作品

私が創作した折り紙作品をご自身で製作した場合、その作品や写真は【二次創作】に準じます。
私が製作した作品(原作)およびその写真は【投稿コンテンツの使用・引用】に準じます。
製作手順が完全に分かるような動画や折り図の公開はご遠慮ください。
投稿にあたって十分に注意しておりますが、他の方の創作物と偶然一致してしまう場合があります。他の方の創作物と同じ作品を見つけた場合、「創作者名」と「作品名」をご連絡ください。
